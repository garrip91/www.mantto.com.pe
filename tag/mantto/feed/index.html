Esta obra forma parte del proyecto “Vive Puerto Viejo” de “Desarrolladora” que estuvo 8 meses en ejecución.

El proyecto está concebido para funcionar como un condominio de playa que contará con 152 casas de playa, un Club House, piscina, áreas de servicio, canchas deportivas y un estacionamiento para 318 automóviles. Además, contará con un sistema de abastecimiento de agua, sistema de desagüe, abastecimiento de energía eléctrica, comunicaciones y sistema de seguridad. [Adjuntamos el video nuestro proyecto]
